# digital-gurukul
08/02/2026

It is a web and mobile based online e-learning and proctored examination platform for any educational institute or industries. mainly there is total 3 type of users (3 screens) for different level & role.

student/learner A student can access all type of study materials like video lectures and short notes , in libraries he/she can access previous year questions , notes , books & any other documents like news etc. a student can take exam and see the result after published the result by teacher/instructor.

teacher/instructor A teacher/instructor can upload the materials(video & short note) in cource section , upload lecture along with a title and description which is contains short notes. A teacher can create exams and published the result. A teacher/instructor can publish notice.

principal/admin (college,school,company & any type of institute) A admin/principal can monitor all users , verify the identification and give the authorization to access there screen based on the role (student/teacher). A admin can publish notice. A admin/principal can monitor all logs of student and teacher (logs are may be anything like login log, exam log & any other suspicious log.) A admin can add more features according to their requirements with the help of developer's team.
